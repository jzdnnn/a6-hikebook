# Folder Gambar Pos Pendakian

Folder ini untuk menyimpan foto-foto pos pendakian yang akan ditampilkan di halaman "Informasi Jalur".

## Gambar yang Diperlukan:

1. **peta-pendakian.png** (di folder /images/)
   - Peta jalur pendakian Gunung Gede Pangrango
   - Rekomendasi ukuran: 1200x800px atau lebih besar
   - Format: PNG atau JPG

2. **pos1-cibodas.jpg**
   - Foto Pos 1 / Basecamp Cibodas
   - Area registrasi dan gerbang masuk

3. **pos2-selabintana.jpg**
   - Foto Pos 2 / Selabintana
   - Area istirahat pertama

4. **pos3-kandangbadak.jpg**
   - Foto Pos 3 / Kandang Badak
   - Area camping utama

5. **pos4-kandangbatu.jpg**
   - Foto Pos 4 / Kandang Batu
   - Pos terakhir sebelum puncak

6. **puncak-gede.jpg**
   - Foto Puncak Gunung Gede
   - Pemandangan dari puncak

## Catatan:
- Semua gambar sebaiknya memiliki kualitas yang baik
- Ukuran file tidak terlalu besar (max 500KB per gambar)
- Format yang disarankan: JPG untuk foto, PNG untuk peta
- Rasio aspek yang bagus: 3:2 atau 16:9
